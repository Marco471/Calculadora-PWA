# Calculadora com Relógio ⏰🖥️

Uma calculadora simples e funcional, com **relógio em tempo real**, feita em **HTML, CSS e JavaScript**.  
Pode ser usada diretamente no navegador ou instalada como **PWA** no celular.

---

## 🚀 Funcionalidades

- Calculadora básica: soma, subtração, multiplicação e divisão  
- Relógio em tempo real com data e dia da semana  
- Responsiva para celular e desktop  
- Código limpo e fácil de entender  

---

## 💻 Tecnologias

- HTML5  
- CSS3 (Flexbox, Gradientes, Responsividade)  
- JavaScript (Variáveis, Arrays, Funções, Eventos)
